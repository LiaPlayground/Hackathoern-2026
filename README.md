## Das Thema

OER-Repositorien wie [OERSI](https://oersi.org) umfassen zigtausende Ressourcen — und trotzdem ist der richtige Treffer oft Zufall. Bestehende Suchlösungen (Volltext, Filter, Graph) kennen keinen Unterschied zwischen einer Lehrenden, die gezielt sucht, und einer Lernenden, die nicht weiß, wonach sie sucht.

Unser Hackathon-Projekt **OER-Navigator** geht einen anderen Weg: Die Darstellung folgt der Intention. Ein LLM-Klassifikator ordnet natürlichsprachige Anfragen einer von 5–8 Suchsituationen zu — und spielt die passende Visualisierung aus. Bei Unklarheiten stellt das System gezielte Rückfragen und macht das Suchprofil sichtbar.

Datenbasis: [OERSI](https://oersi.org) (~96.000 Ressourcen, offene Elasticsearch-API, AMB-Metadaten).
